## 📋 說明

這是一個半封閉型的 FAQ 機器人，只會回答 `faq.json` 裡面有的問題，不會亂講。

**適用對象**：
- 👩‍⚕️ 診所員工（快速查詢標準答案）
- 👥 客戶/病人（自助查詢常見問題）

---

## 🚀 使用方式

### 本地測試
```bash
cd 血液淨化FAQ機器人
python3 -m http.server 8000
# 然後打開 http://localhost:8000
```

### 部署到 GitHub Pages
1. 建立 GitHub Repository
2. 上傳這個資料夾的所有檔案
3. 設定 > Pages > 選擇 main branch
4. 等幾分鐘就可以用了

---

## 📁 檔案結構

```
血液淨化FAQ機器人/
├── index.html    ← 網頁介面
├── faq.json      ← FAQ 資料（要改內容改這裡）
└── README.md     ← 說明文件
```

---

## ✏️ 如何更新 FAQ

編輯 `faq.json` 檔案：

```json
{
  "id": 9,
  "keywords": ["新關鍵字1", "新關鍵字2"],
  "question": "新問題？",
  "answer": "完整回答...",
  "shortAnswer": "一句話回答"
}
```

**欄位說明**：
- `id`：編號（不重複）
- `keywords`：觸發關鍵字（使用者輸入包含這些字就會觸發）
- `question`：問題標題
- `answer`：完整回答
- `shortAnswer`：一句話簡答
